"Hey, I'm Geri! I like to play chess against myself but I can also help you setup a tournament if you'd like.
 
Tourney Commands: 
-- $maketourney : allows you to create a Round Robin tournament with a list of players. (Each player will play each other twice.) 
-- $stoptourney : ends the current tournament 
-- $tourney : gives the current tournament info
-- $players : lists all the players in a tournament 
-- $standings : lists the current tournament score count 
-- $addwinner : allows you to add a score to an ongoing game 
-- $addli : adds a lichess username to a current tournament player's info
-- $lastli : shows your last game(based on start date, not end date of the game)
Misc. Commands: 
-- $puzzle : generates a chess puzzle in the chat 
-- $findli : will find a lichess games using usernames
-- $resources : provides a list of useful chess resources 

I'm old so go easy on me!"
